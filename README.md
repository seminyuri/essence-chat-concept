# Суть ✕ Чат — концепт мессенджера

Интерактивный **фронтенд-концепт** внутреннего мессенджера экосистемы «Суть»
(Суть · Суши Мама · Изумрудиум + клиенты + внешние контакты).
Плейсхолдер-данные, без бэкенда — но всё кликается. Дизайн-язык Essence:
тёплый rose-gold, Manrope / Lora italic / JetBrains Mono, hairline-бордеры,
светлая и тёмная темы.

**Живая версия:** открой `index.html` или ссылку GitHub Pages. Vanilla JS, ноль зависимостей.

## Что внутри
- **Мультиорг-воркспейсы** — Суть · Суши Мама · Изумрудиум · Личное, у каждой свой акцент (переключатель вверху списка).
- **Типы чатов** — личка, группы (с ролями), каналы-broadcast, **клиентские** (с натал-онбордингом), **внешние** (поставщики через MAX/VK), Избранное.
- **Сообщения** — реакции, «кем прочитано», ответы, пересылка, медиа, файлы, **голосовые с авто-расшифровкой**, упоминания, service-строки, day-separators, галочки доставки/прочтения.
- **Композер** — эмодзи-пикер, вложения, голос, ответ/редактирование.
- **Инфо-панель** — профиль, участники + роли (владелец/админ/гость), **натал-карта клиента**, медиа/файлы/ссылки/голос.
- **Оверлеи** — ⌘K-палитра, контекст-меню сообщения, быстрые реакции, форвард-пикер, меню создания.
- **Дифференциатор** — приглашение по ссылке с **тумблером «запрашивать дату/время/место рождения»** (для клиента — да, для поставщика — нет) + **лендинг входа клиента** с онбордингом или «упрощённо».

## Deep-links (состояния для просмотра)
| Параметр | Значение |
|---|---|
| `?ws=` | `sushi` · `izum` · `personal` — организация |
| `?chat=` | `maria` (клиент) · `team` (группа) · `supply` (внешний) · `announce` (канал) · `anya` … |
| `?panel=1` | открыть инфо-панель |
| `?theme=dark` | тёмная тема |
| `?view=join&req=1` | лендинг входа клиента (с натал-данными) |
| `?view=join&req=0` | лендинг входа внешнего (без натал-данных) |
| `&step=natal` / `&step=done` | шаг лендинга |

> Концепт-прототип. Не для продакшена — плейсхолдеры вместо бэкенда.
