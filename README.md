# Суть ✕ Чат — концепт мессенджера

Интерактивный **фронтенд-концепт** внутреннего мессенджера экосистемы «Суть»
(Суть · Суши Мама · Изумрудиум + клиенты + внешние контакты).
Плейсхолдер-данные, без бэкенда — но **всё кликается**. Дизайн-язык Essence:
тёплый rose-gold, Manrope / Lora italic / JetBrains Mono, hairline-бордеры,
светлая и тёмная темы. Vanilla JS, ноль зависимостей.

**Живая версия:** открой `index.html` или ссылку GitHub Pages.

## Что внутри
- **Мультиорг-воркспейсы** — Суть · Суши Мама · Изумрудиум · Личное; у каждой свой акцент (переключатель вверху списка).
- **Филиалы/подразделения** — у Суши Мамы папки-вкладки: Центр · Юг · Управление · Поставщики.
- **Типы чатов** — личка, группы (роли, права), каналы-broadcast (просмотры), **клиентские** (натал-онбординг), **внешние** (поставщики), Избранное.
- **Сообщения** — реакции + эмодзи-пикер, **«кем прочитано»**, ответы, пересылка, медиа, файлы, **голосовые с авто-расшифровкой**, **опросы**, стикеры, линк-превью, упоминания, service-строки, галочки, черновики, day-separators.
- **Мощь чата** — закреплённые сообщения, поиск внутри чата с подсветкой, мультивыбор + массовые действия, опции отправки (без звука / позже).
- **Композер** — эмодзи, вложения (реально отправляют), голос, ответ/редактирование.
- **Звонки** — аудио и видео, 1:1 и групповые (сетка тайлов), живой таймер, mute/камера/динамик.
- **Инфо-панель** — профиль, участники + роли, **натал-карта клиента**, медиа/файлы/ссылки/голос.
- **Управление группой** — тип (приватная/публичная), «кто что может» (5 прав), медленный режим, роли.
- **Оверлеи** — ⌘K-палитра, контекст-меню, быстрые реакции, форвард-пикер, уведомления-дропдаун, меню создания.
- **Создание** — личка (пикер контактов), группа (2 шага: участники → название/цвет), канал; кадрирование аватара.
- **Настройки** — профиль, оформление (**живые** тема/акцент/плотность), уведомления, приватность («кто видит»), устройства, организации.
- **Дифференциатор** — приглашение по ссылке с **тумблером «запрашивать дату/время/место рождения»** + **лендинг входа клиента** (Telegram/VK/MAX → онбординг или «упрощённо»).
- Светлая + тёмная темы, мобайл.

## Deep-links (для просмотра состояний)
| Параметр | Значение |
|---|---|
| `?ws=` | `sushi` · `izum` · `personal` — организация |
| `?chat=` | `maria` (клиент) · `team` (группа+опрос+пин) · `announce` (канал) · `supply` (внешний) · `anya` (стикер) … |
| `?panel=1` | инфо-панель · `?theme=dark` тёмная тема |
| `?call=audio` / `?call=video` | звонок в открытом чате |
| `?view=join&req=1` | лендинг входа клиента (с натал) · `&req=0` без натал · `&step=natal\|done` шаг |
| `?view=settings&sec=` | `profile` · `appearance` · `notifications` · `privacy` · `devices` · `orgs` |

> Концепт-прототип. Не для продакшена — плейсхолдеры вместо бэкенда.
